# ShutterBid App Store Listing Draft

## Name
ShutterBid

## Subtitle / Short description
Photography job marketplace beta.

## Long description
ShutterBid is a beta marketplace foundation from MackSims for clients, photographers, and business/venue partners. The app is being developed to support photography job posting, profile setup, bidding workflows, and marketplace coordination.

During beta, testers can help validate account flows, job creation, photographer profiles, bidding paths, partner concepts, mobile layout, and support workflows.

## Important beta limitation
ShutterBid is in beta. Do not rely on it for guaranteed bookings, real payments, verified professionals, background checks, live moderation, dispute resolution, or business-critical outcomes unless the current build clearly states those features are live.
