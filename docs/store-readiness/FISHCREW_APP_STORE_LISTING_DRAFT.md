# FishCrew App Store Listing Draft

## App Name

FishCrew

## Subtitle / Short Description

Fishing community and water tools.

## Long Description

FishCrew is a MackSims beta fishing community app with browse-first behavior, profile testing, water-tool
surfaces, and the approved Port Window design direction. Testers can review the app shell, navigation, profile
state, and community surfaces while the team prepares native wrapper and store-readiness materials.

During beta, some content, water tools, profile states, or feed surfaces may be mock, demo, or preview content.
GPS, native device permissions, real-time location, paid features, and production safety systems are not part of
the current beta web shell unless a specific native build says otherwise.
