# ShutterBid App Store Listing Draft

## App Name

ShutterBid

## Subtitle / Short Description

Photo work marketplace for clients and pros.

## Long Description

ShutterBid is a MackSims beta marketplace foundation for clients, photographers, and business or venue partners.
The app is designed to help testers review role-aware profiles, feed surfaces, portfolio paths, and early bidding
structure for visual work.

During beta, some marketplace content, jobs, bids, profiles, or review surfaces may be mock, demo, or preview
content. Production payments, contracts, escrow, dispute handling, verified marketplace transactions, and live
moderation should not be assumed unless a specific build says those systems are connected and ready.
