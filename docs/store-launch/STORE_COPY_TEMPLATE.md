## Copy Safety Checklist

- [ ] Describes the submitted build.
- [ ] Avoids unconfirmed partnerships.
- [ ] Avoids unconfirmed payments, dispatch, brokerage, safety, health, financial, legal, or moderation claims.
- [ ] Avoids implying guaranteed outcomes.
- [ ] Mentions demo/preview limitations when relevant.
- [ ] Matches screenshots and release notes.
