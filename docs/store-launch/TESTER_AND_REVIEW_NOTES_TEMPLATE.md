- [ ] App opens.
- [ ] Primary navigation works.
- [ ] Login/logout works if applicable.
- [ ] Account/profile state is correct if applicable.
- [ ] Core workflow works.
- [ ] Support/privacy/account deletion links work.
- [ ] No private test data is visible.
- [ ] No unfinished production claims are visible.
- [ ] App handles no-network or error state acceptably.
