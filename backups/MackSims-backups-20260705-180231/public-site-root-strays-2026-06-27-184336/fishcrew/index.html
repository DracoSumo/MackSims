FishCrew

Beta fishing community and water tools app.

FishCrew is a browse-first fishing community app with mobile navigation, profile testing, water-tool surfaces, and the approved Port Window direction.

Tester focus

Verify Home, Explore, Crew, Feed, Tools, Tools Guide, Profile, and account switching. Watch for stale local profile state after logout and second-account sign-in.

Known limitations

GPS, native device permissions, real-time location, paid modules, and production safety features are outside this beta web shell pass unless a specific native build says otherwise.
